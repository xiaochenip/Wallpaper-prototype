# 壁纸样机生成器

一个在线工具，允许用户上传壁纸并应用到各种设备模板上，生成具有真实感的展示效果。

## 功能特性

- 支持多种设备模板（手机、平板、笔记本电脑）
- 拖拽上传壁纸图片
- 实时预览效果
- 自定义背景和缩放
- 一键下载生成的样机图片

## 安装和运行

### 前端版本（静态）
1. 下载所有文件到你的web服务器目录
2. 确保`assets/images/templates/`目录中有设备模板图片
3. 在浏览器中打开`index.html`

### 完整版本（Node.js）
1. 确保已安装Node.js
2. 运行 `npm install` 安装依赖
3. 运行 `npm start` 启动服务器
4. 访问 `http://localhost:3000`

## 使用说明

1. 点击上传区域或拖拽图片上传壁纸
2. 选择设备模板
3. 调整设置（背景颜色、缩放等）
4. 点击下载按钮保存生成的样机图片

## 技术栈

- 前端: HTML5, CSS3, JavaScript, Fabric.js
- 后端: Node.js, Express, Sharp, Multer
